**Description**  
A cross-platform desktop application designed to allow users to track fianical goals, set and follow budgets and ease expense tracking.

#### Goal - The Big Why

Objective: To allow a easy UI to track a budget that is cross compatible with devices. This is designed to be a local first data setup to keep the data private.

Target: Myself... And anyone who would like a simple budget app that takes less effort than an excel spreadsheet but offers similar fuctionality.

#### Problem/Solution

Problem: Everything requires the internet to even load an app nowdays. Leaving data exposed and users reliant upon a connection.

Solution: A local first app that allows most if not all of the functionality to be done straight from the users device.
